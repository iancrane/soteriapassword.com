Windows Updates

Soteria Password uses the latest technology to provide fast and secure access to your data, which makes it very important to keep up-to-date on Windows Updates. If you experience any crashes, that is an indication that you are missing one or more updates.

It may take more than one try to get all of the updates you need. In such cases, Windows Update will show that there are no more updates, but if you click Check for Updates, more will magically appear.

Click Start / All Programs / Windows Update
