Soteria Password™

Accelerators & Shortcuts

Soteria offers generous keyboard accelerators and shortcuts to empower the folks who ain't got time for chasing mice.

Accelerators

Accelerators use the Alt key in combination with other keys to select specific input fields.

Note - pressing Alt reveals all available accelerators by underlining the corresponding key in each label.

KeysDescription
Alt+O Focus the Open field
Alt+U Focus the User field
Alt+P Focus the Password field

Shortcuts

Below is the list of Soteria commands that can be run using keyboard shortcuts:

KeysCommand
Ctrl+B Backup passwords database
Ctrl+D Add Entries
Ctrl+Del Delete Entry
Ctrl+E Edit Entry
Ctrl+F Find Entries
Ctrl+G Password Generator
Ctrl+H Hide Notes
Ctrl+L Log In as a Different User
Ctrl+M Show menu
Ctrl+O Open a passwords database
Ctrl+P Copy Password
Ctrl+S Save changes
Ctrl+U Manage Users and Groups
Ctrl+W Change Owner
Enter Open Entry
F5 Refresh Entries List
