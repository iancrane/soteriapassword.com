Soteria Password™

Go Super Fast

The Find box provides a lightening-fast method to find your target entry via the keyboard.

Soteria automatically filters the list as you type, only showing entries with matching text in any field (other than Password).

Simply start typing any part of your target entry into the Find box until it is selected, and then press Enter.

Note - Pressing Enter does the same thing as clicking .

Example

  1. Press Ctrl+F to focus the Find box.
  2. Type X and press Enter to select Netflix.

Explanation

This example works because:

  • The Find box only lists matching entries.
  • There is only one entry that contains an X, so that is all you need to select the entry.
