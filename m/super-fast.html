Go Super Fast

Start typing text that matches any part of your target entry into the Find box to filter the list for matching results, and then press Enter when your entry is highlighted (or click ).

Simply start typing any part of your target entry into the Find box until it is selected, and then press Enter (or click ).

Example

  1. Press Ctrl+F to focus the Find box
  2. Type X and press Enter (or click ) to go to Netflix

Explanation

This example works because:

  • The Find box only lists matching entries
  • There is only one entry that contains an X
  • Note that pressing Enter is the same as clicking
